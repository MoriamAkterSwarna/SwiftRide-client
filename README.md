# SwiftRide Client

## Key Points
- Built with React, TypeScript, and Vite
- Modern, responsive UI for ride-sharing platform
- Integration with backend API for all features
- Google OAuth login
- Real-time ride status and driver tracking
- Payment flow integration
- Role-based dashboards (User, Driver, Admin)
- Error boundaries and skeleton loaders for UX

## Key Functionalities
- User registration, login, and profile management
- Ride booking and fare estimation
- Live map and driver location updates
- Payment and ride history
- Admin panel for managing users, drivers, and rides
- Review and rating system
- SOS and helpline features
